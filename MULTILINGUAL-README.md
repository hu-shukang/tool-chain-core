### 1. 快速示例增强

所有三个语言版本的 30 秒快速示例都已改进，包含：

```typescript
// 每一步都有详细注释说明 r1、r2 的含义
const result = await new Chains()
  .chain(() => 10) // 第1步：返回 10，r1 = 10
  .chain((r) => r.r1 * 2) // 第2步：r1=10，返回 20，r2 = 20
  .chain((r) => r.r2 + 5) // 第3步：r2=20，返回 25，r3 = 25
  .invoke();
```

关键概念说明：

- ✅ r1 是第1步的结果
- ✅ r2 是第2步的结果
- ✅ 每步可以访问所有前面的结果
